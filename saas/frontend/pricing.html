How LAN Network Sync works

Share patient data across workstations on your local network — no cloud, no latency, no GDPR risk.

🖥️

1. Set up a shared folder

Point PISUM to a shared network folder (NAS, SMB share, or mapped drive). All workstations on the same network access the same database.

🔐

2. One shared password

The database is AES-256 encrypted. Set one password on the first workstation — all other PCs enter the same password to connect.

⚡

3. Instant sync across PCs

Each workstation reads and writes directly to the shared database. Changes appear instantly across all connected computers.
