Installation

Pisum est une application de bureau légère conçue pour Windows. Elle ne nécessite aucune configuration informatique complexe.

  1. Téléchargez le dernier installateur via le lien fourni dans votre e-mail de bienvenue.
  2. Double-cliquez sur le fichier .exe téléchargé.
  3. Suivez les instructions à l'écran. Les droits d'administrateur ne sont généralement pas requis.
Note sur la confidentialité

Pisum fonctionne entièrement hors ligne par défaut. Les données des patients ne sont jamais envoyées sur un serveur cloud.

Utiliser les modèles

Pisum est livré avec 112 modèles experts couvrant 5 modalités.

Sélectionner un modèle

Dans le panneau de gauche, naviguez dans les dossiers par modalité ou utilisez la barre de recherche.

Modèles personnalisés

Vous pouvez modifier n'importe quel modèle intégré ou créer le vôtre. Ils sont sauvegardés localement.

Dictée vocale IA

Le moteur intégré Sally AI vous permet de dicter en mains-libres dans 23 langues.

  • Cliquez sur l'icône 🎙️ ou appuyez sur F4.
  • Sélectionnez votre langue préférée dans la barre de dictée.

Export RIS/PACS

Pisum est conçu pour agir comme un outil compagnon sans intégrations complexes.

Copier-Coller (Recommandé)

Cliquez sur 'Copier le rapport'. Le formatage est préservé et prêt à être collé.

Export Word

Utilisez 'Exporter en DOCX' pour générer un document formaté.
