01 Installation

Pisum est une application de bureau légère conçue pour Windows. Elle ne nécessite aucune configuration informatique complexe.

  1. Téléchargez le dernier installateur via le lien fourni dans votre e-mail de bienvenue.
  2. Double-cliquez sur le fichier .exe téléchargé.
  3. Suivez les instructions à l'écran. Les droits d'administrateur ne sont généralement pas requis.
Note sur la confidentialité Pisum fonctionne entièrement hors ligne par défaut. Les données des patients ne sont jamais envoyées sur un serveur cloud.

02 Utiliser les modèles

Pisum est livré avec 112 modèles experts couvrant 5 modalités (Scanner, IRM, Échographie, Radiographie, Interventionnel).

Sélectionner un modèle

Dans le panneau de gauche, naviguez dans les dossiers par modalité (ex: Scanner > Abdomino-Pelvien) ou utilisez la barre de recherche en haut pour trouver rapidement un protocole spécifique.

Modèles personnalisés Vous pouvez modifier n'importe quel modèle intégré ou créer le vôtre en cliquant sur 'Nouveau Modèle'. Ils sont sauvegardés localement sur votre profil.

03 Dictée vocale IA

Le moteur intégré Sally AI vous permet de dicter vos comptes rendus en mains-libres dans 23 langues, en corrigeant automatiquement les termes radiologiques.

  • Cliquez sur l'icône 🎙️ ou appuyez sur F4 pour démarrer/arrêter l'enregistrement.
  • Sélectionnez votre langue préférée dans le menu déroulant de la barre de dictée.

04 Export RIS/PACS

Pisum est conçu pour agir comme un outil compagnon qui alimente facilement vos systèmes existants sans intégrations complexes.

Copier-Coller (Recommandé)

Cliquez sur 'Copier le rapport' (ou Ctrl+C) une fois votre rapport terminé. Le formatage du texte est préservé et prêt à être collé directement dans l'éditeur de votre RIS ou PACS.

Export Word

Utilisez le bouton 'Exporter en DOCX' pour enregistrer un document formaté professionnellement, idéal pour les cliniques privées remettant directement les rapports aux patients.
